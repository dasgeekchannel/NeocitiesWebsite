My Favorite Linux Software

This is a list of my favorite software applications for Linux. I hope this list helps you discover some new and amazing software that runs on Linux. I've provided links to the project webpages for more information. Installation of the software should be done via your distros repository such as Ubuntu (sudo apt install Kdenlive) or Arch (pacman -S kdenlive). Some of the software is available as a SNAP or Flatpak so check for universal package options if it is not in your distro repository or software store.
